# portfolio-vue

This is a Portfolio website that was done using Vue
It contains work Experience, Projects and Contact information. 

Live site: 
[anthonyruiz.dev](https://anthony-ruiz.github.io/portfolio/)


## Tools used:
* Vue
* Vue Router
* VueX
* Bootstrap Vue
* JavaScript
* HTML
* CSS
